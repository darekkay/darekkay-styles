Prism.js

Prism.js is supported out of the box. Here's some CSS code:

@import "manual.css";

h1.mystyle[href^="http"]:lang(en) {
  color: blue; /* TODO: change THIS to yellow for next version! */
  border: rgb(255,0,0);
  background-color: #FAFAFA;
  background: url(hello.jpg) !important;
}

And here's some pro JavaScript code:

function log(message) {
  console.log("Log:", message.replace(/^\s*(.*)/, "moo"));
}
